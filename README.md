### Overview
Fabric script loader for Minecraft java Edition. Make fabric loader support JS scripting like Bedrock Edition.  
This mod is available on [Modrinth](https://modrinth.com/) **(but, not for now)**

> Dependency:
> - Fabric Api

### Building
building mods from source
```
# clone this repository
git clone https://github.com/rmz3t-dev/z3-script-loader

# goto mods source directory
cd z3-script-loader/src
./gradlew build
```
